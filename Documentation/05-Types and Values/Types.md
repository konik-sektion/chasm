# Types and Values

## Primitive Types

| Type | Size | Description |
|------|------|-------------|
| u8 | 1 B | Unsigned 8-bit |
| u16 | 2 B | Unsigned 16-bit |
| u32 | 4 B | Unsigned 32-bit |
| u64 | 8 B | Unsigned 64-bit |
| i8 | 1 B | Signed 8-bit |
| i16 | 2 B | Signed 16-bit |
| i32 | 4 B | Signed 32-bit |
| i64 | 8 B | Signed 64-bit |
| Null | 0 | No value |

## Typepointing

```chasm
#section bss
let x:u8;
let y:i32;
```

# Introduction

## What is "ChASM"?

ChASM is a low-level, systems programming language that assembles directly to NASM-
compatible x86_64 ASM.

It is namely designed for:
- OSdev;
- Bootstrapping runtimes;
- Performance-critical components; and,
- Low-level tooling.

## What `is not` ChASM

- ChASM is not GC'd.
- ChASM is not object-oriented.
- ChASM is not abstract.
- ChASM is not implicit.
